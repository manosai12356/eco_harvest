# EcoHarvest

## Description
EcoHarvest is a blockchain and IPFS-based platform designed to ensure transparency and trust in organic farming. The system allows users to verify organic products through QR codes linked to immutable blockchain records and decentralized storage. This helps prevent fraud, improves traceability, and builds consumer confidence in organic produce.

## Features
- QR code-based product verification
- Blockchain-backed data integrity
- IPFS storage for farm records and media
- Transparent supply chain tracking
- User-friendly web interface
